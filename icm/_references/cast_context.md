# Cast Context

## Purpose

This file stores reusable character context that may help resolve identities and interpret panels. It should be updated only after human review.

Do not use this file to invent identity. Use it to support identity when the panel evidence is compatible.

## Confidence Rule

Use these confidence levels:

- `high`: supported by clear visual evidence, dialogue, or repeated confirmed context
- `medium`: likely based on visual/contextual evidence, but not fully certain
- `low`: possible but weakly supported
- `uncertain`: do not assign a proper name

## Character Entries

### Toph

Known traits:

- blind
- perceives the world primarily through earthbending/seismic sense rather than ordinary sight
- often used for jokes involving visual misunderstanding or nonvisual perception

Interpretation notes:

- If a panel depends on seeing fireworks, colors, faces, distant objects, or written details, check whether Toph's blindness changes the meaning.
- If Toph describes what something "sounds like" or reacts to impact/vibration, treat that as potentially intentional setup/payoff.

Visual identifiers:

- often associated with green clothing
- dark hair
- small stature compared to older characters

Status: seed context, needs expansion from reviewed chapters.

## Update Rule

When a character is confirmed from a reviewed chapter, add:

- visual identifiers
- dialogue patterns if useful
- known relationships

Do not bloat this file with one-off descriptions unless they help future identity resolution.
